# Code of Conduct

This project adopts the **Contributor Covenant Code of Conduct, version 2.1**.

Full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In short: be respectful, welcoming, and constructive. Harassment, trolling,
personal attacks, and other unprofessional conduct are not tolerated.

## Reporting

Report violations to **chris@ForbesAssetManagement.com**.

All reports are handled confidentially. The maintainers will review, respond
within 3 business days, and take action proportional to the violation —
ranging from a private warning up to a permanent ban.

## Scope

This Code of Conduct applies within all project spaces (GitHub issues,
discussions, PRs, Discord if any) and when an individual is representing the
project in public spaces.
